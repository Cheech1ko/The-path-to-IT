# 💈 Barber Shop Website

Лендинг для барбершопа с современным дизайном и функционалом записи онлайн.

## 🚀 Функционал
- Приветственный экран с услугами
- Галерея работ
- Online-запись
- Контактная информация
- Адаптивный дизайн

## 🛠️ Технологии
- HTML5
- CSS3 (Flexbox, Grid, анимации)
- JavaScript (форма записи)
- Git
